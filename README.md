# my_scripts
Useful shell scripts for me.

Overview:

| Script Name  | Descrption                                      |
|:-------------|:------------------------------------------------|
| makeAppAlias | MacのAppをターミナルで開くためのaliasを作成する |
| my_rm        | rmコマンドでゴミ箱に移動する                    |


Install:
```sh
$ git clone https://github.com/rednes/my_scripts.git
$ ./scripts/install.sh
```
